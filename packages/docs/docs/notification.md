# Notification

## Props

| 属性名           | 类型          | 默认值    | 描述                        |
|------------------|---------------|-----------|-----------------------------|
| visible          | `boolean`     | -         | 控制消息显示。              |
| title            | `string`      | `'标题'`  | 消息标题。                  |

## 事件

| 事件名           | 参数类型     | 描述                                       |
|------------------|--------------|--------------------------------------------|
| update:visible   | `boolean`    | 当消息框的可见性发生变化时触发。             |

## 插槽

- 默认插槽：消息框的内容。
