# Radio

单选框组件，支持选中状态切换。

unique属性用来标记单选框组，name属性用于区分组内不同单选框，也是组件框的值，label是组件框的标签。

muti可以开启多选

## Props

| 属性名           | 类型          | 默认值      | 描述                                       |
|------------------|---------------|-------------|--------------------------------------------|
| muti             | `boolean`     | `false`     | 是否支持多选。                             |
| name             | `string`      |             | 单选框的名称。                             |
| label            | `string`      |             | 单选框的标签。                             |
| unique           | `string`      |             | 唯一标识符。                                |
| value            | `boolean`     | `false`     | 单选框的选中状态。                         |
| type             | `'default'`<br>`'primary'` | `'default'` | 单选框的类型。                             |

## 事件

| 事件名            | 参数类型     | 描述                                       |
|-------------------|--------------|--------------------------------------------|
| update:value      | `boolean`    | 选中状态发生变化时触发。                     |

## 插槽

无。
