# Switch

切换栏组件，用于在不同选项之间切换。

## Props

| 属性名           | 类型              | 默认值        | 描述                                           |
|------------------|-------------------|---------------|------------------------------------------------|
| switcherOptions  | `switcherOptionsType` |             | 切换栏选项。`switcherOptionsType` 是一个自定义类型，表示切换栏的选项。 |
| value            | `string`          |             | 切换栏当前指向的 key。                            |
| type             | `'default'`<br>`'tag'` | `'default'` | 切换栏的样式类型。                                |

## 事件

| 事件名            | 参数类型           | 描述                                      |
|-------------------|--------------------|-------------------------------------------|
| update:value      | `string`           | 切换值发生变化时触发。                        |

## 插槽

无。
