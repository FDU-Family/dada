# PopOut

## Props

| 属性名           | 类型          | 默认值      | 描述                                       |
|------------------|---------------|-------------|--------------------------------------------|
| visible          | `boolean`     | `false`     | 控制组件弹出。                             |
| mask             | `boolean`     | `true`      | 是否显示遮罩。                             |
| maskClosable     | `boolean`     | `true`      | 点击遮罩后是否可以关闭。                    |

## 事件

| 事件名            | 参数类型     | 描述                                       |
|-------------------|--------------|--------------------------------------------|
| update:visible    | `boolean`    | 当弹出组件的可见性发生变化时触发。             |

## 插槽

- 默认插槽：弹出组件的内容。
